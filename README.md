# DExT: Detector Explanation Toolkit

## Installation
```
pip install -e .
```

## Object detectors with weights available
1. EfficientDet - COCO
2. SSD512 - COCO

## Interpretation methods available
1. Intergrated Gradients (IG)
2. Guided Backpropagation (GBP)
3. SmoothGrad (SG) with IG and GBP
